Create a proactive dashboard.

This idea is from Noah Kagan, the founder of AppSumo. Essentially, this is a measurable task list. If you’re noticing that a certain task is having a big effect on our business or is a necessary task that you need to keep doing, then list out the number of times that you need to do that task each week. The goal here is to record inputs, not just outcomes. This is especially useful for marketers. If you know you need to tweet or write a certain number of LinkedIn posts, then you can list these out and make sure you do that number for the week. Another way to use a proactive dashboard is using your team’s project management software during team meetings to track progress on projects. There are plenty of alternatives out there including Gantt Chart software.
